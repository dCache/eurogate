Eurogate Installation in Simulation Mode

Installing the package

The EuroGate Simulation package distinquishes between the PackageRoot and the dataRoot denoted as <packageRoot> and <dataRoot> resp.. The package root is the directory where the package classes, binaries and scripts are inflatted and the data root will contain all necessary databases and the simulated tapes. Consequently the data root need appropriate space to store the data. 
cd <packageRoot>
tar xvf /cdrom/eurogate.sim.tar
cd eurogate/jobs
cp eurogateSetup.template eurogateSetup
  • Find a partition on the local host which sufficient space to store the simulated data tapes, and create a subdirectory. ( will be called <dataRoot> )
  • Find the entry databaseRoot= in eurogateSetup and assign the variable to the full path of this subdirectory.
  • Create the following subdirectories : 
        <dataRoot>/store/raw
    <dataRoot>/store/dst
    <dataRoot>/pvr/stk
    <dataRoot>/pvl
    <dataRoot>/mover
  • Create the login ssh keys in <packageRoot>/eurogate/jobs 
        ../bin/ssh-keygen -b  768 -f ./server_key -N \"\"
    ../bin/ssh-keygen -b 1024 -f ./host_key   -N \"\"
  • Run ./eurogate start

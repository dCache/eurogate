Eurogate Installation in Simulation Mode

Installing the package

  • Installation from CD

    The installation CD contains an installation script 'install.sh' which does most of the work. The software goes to /opt/EurogateV7 by default, or any other directory if specified otherwise. 

    cd /cdrom/hsm
./install.sh
  • Installation from eg.tar file

    Create the sourceRoot directory. The default for this version is /opt/EurogateV7 and deflate the eg.tar file into this directory. 

    mkdir -p /opt/EurogateV7
cp ???/eg.tar /opt/EurogateV7
cd /opt/EurogateV7
tar xf eg.tar

Database Preparation

In simulation mode, the data files are stored on the local filesystem. Therefore there must be suffient disk space to hold the datafiles. When starting the SimulationSystem the first time, all necessary initialization is done automatically. The administrator has to supply the system with an empty subdirectory. (The <databaseRoot>) 
mkdir -p <databaseRoot>
  • Customizing the setup file

    • Change to the <eurogate>/jobs directory.
    • copy eurogateSetup.template to eurogateSetup.
    • Customize the content of eurogateSetup. The setupfile contains key value pairs which determine the behaviour of the system. Some of the values are already set to a reasonable value, others MUST be tuned to fit the site specific needs.
      • databaseRoot must be set to the directory path prepared above. When the system is started the first time, this subdirectory must be empty. It must provide sufficent disk space to hold the datafile, simulating the tapes.
      • java If 'java' is not set, the system assumes to find the java binary in the PATH variable.

    • Starting/Stopping the system

    The system is started and stopped from within the 'jobs' directory. 
    cd ../jobs
./eurogate start

resp.

./eurogate stop

    Administering the system

    The system provides a graphical and a line interface for most administative and monitoring tasks. For the command line interface it uses the ssh Version I protocol for login. Both access methods are started from within the eurogate jobs directory

    • Command Line 
      ./eurogate admin -l admin
      The initial password is 'dickerelch'

    • Graphics 
      ./eurogate gui
      The login user name has to be changed to 'admin' and the initial password is 'dickerelch'.

      See EuroGuide for administration commands.

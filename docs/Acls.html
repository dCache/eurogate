The EuroStore Acl's

User Administration

CommandAclComment
create user <user> user.*.create creates a user entry
create group <group> user.*.create creates a group entry
destroy principal <principal> user.*.create destroys a user/group
add <principal> to <group> user.<group>.access adds a user/group to a group
set <principal> [<key>=<value> ...] user.<principal>.access sets the attributes of a user/group
show groups - shows all groups
show group <group> - shows the members (users/groups) of <group>
show principal <principal> - shows attributes of <group>

Acl Administration

CommandAclComment
create acl <aclName> super.access creates an acl entry
destroy acl <aclName> super.access destroys an acl entry
show acl <aclName> - show the members of this acl
add access [-allowed|-denied] <acl> <principal> acl.<acl>.access add a principal to this acl (allow or deny)

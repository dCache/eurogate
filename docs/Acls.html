The EuroStore Acl's

User Administration

CommandAcl
create user <user> user.<user>.create
create group <group> user.<user>.create
destroy principal <principal> user.<principal>.destroy
add <principal> to <group> user.<group>.add
remove <principal> from <group> user.<group>.remove
set <principal> [<key>=<value> ...] user.<principal>.modify
show groups -
show group <group> -
show principal <principal> -
set passwd user.<principal>.changePassword

Acl Administration

CommandAcl
create acl <aclName> acl.<aclName>.create
destroy acl <aclName> acl.<aclName>.destroy
show acl <aclName> -
add access [-allowed|-denied] <acl> <principal> acl.<acl>.add
remove access <acl> <principal> acl.<acl>.remove
let <destinationAcl> inheritfrom <sourceAcl> acl.<destinationAcl>.modify

Pvl Administration

Some Remarks :

  • Eurogate only supports a single 'pvl'. The name of this pvl is always pvl. This may change in the future. The 'acl' scheme already supports multiple pvls, mainly to support the use of one pvr by multiple pvls.
  • It is assumed that the names of the pvrs are unique. This must still be true in case multiple pvls are supported.
CommandAcl
create pvr <pvrName> pvl.<pvlName>.admin
create cartridgeDescriptor <name> pvl.<pvlName>.admin
create volumeDescriptor <name> pvl.<pvlName>.admin
create volume -pvr=<pvrName> <volumeName> pvl.<pvlName>.<pvrName>.support
set volume <volumeName> pvl.<pvlName>.<pvrName>.operate
create volumeSet <name> pvl.<pvlName>.support
add volume <volumeName> -vs=<volumeSet> vs.<pvlName>.<volumeSet>.support
create drive -pvr=<pvrName> <driveName> pvr.<pvrName>.admin
create cartridge -pvr=<pvrName> <volumeName> pvr.<pvrName>.operate
set drive -pvr=<pvrName> <driveName> ... drive.<pvrName>.<driveName>.support
enable/disable/dismount <driveName> drive.<pvrName>.<driveName>.operate
update drive.<pvrName>.<driveName>.operate
